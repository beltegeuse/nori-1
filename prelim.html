Cloning using the command line

The source code of Nori is available on github. For details on git in general, refer to github's Introduction to Git. To check out a copy of the source code, enter 

$ git clone --recursive
	https://github.com/wjakob/nori2

on the terminal. It is a good habit to frequently commit changes you made to the repository; this will allow you to jump back and forth between different versions, which can be quite useful for debugging. Note that we'll publish extensions and bug-fixes to this repository as the semester progresses. Run the command 

$ git pull
to merge them into your repository and benefit from the improvements.

Cloning using source tree

On Windows and Mac OS X, Source Tree is a convenient visual front-end to Git. We particularly recommend this tool if you've never used Git before. When cloning the project, make sure that the setting for recursive cloning is active under Advanced Options.
